## @{keyword.statement}

``` python
def fib(n: int) -> int:
    if n <= 1:
        return n
    else:
        return (fib(n - 2) + fib(n - 1)) % 1000000007
```

## @{keyword.constraints}

- $0 \leq n \leq 10^{18}$

## @{keyword.input}

```
$n$
```
