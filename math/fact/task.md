## @{keyword.statement}

``` python
def fact(n: int) -> int:
    if n == 0:
        return 1
    else:
        return n * fact(n - 1) % 1000000007
```

## @{keyword.constraints}

- $0 \leq n \leq 10^{7}$

## @{keyword.input}

```
$n$
```
